# To Do App

The classic To Do app.

# Live

[Go Live](https://to-react-do.herokuapp.com/)


# Pre-requisites

- Node.js v16.13.1

You can install dependencies with `npm install` or `yarn install`.

Then you can run the project with  `npm start` or `yarn start` command.

# Basic Knowledge of

- React
- React Hooks
